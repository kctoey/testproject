
# CRUD project 

This project is for interview test.
Use API from "https://jsonplaceholder.typicode.com/users"


## Features

- Create employee data
- Read employee data
- Update employee data
- Delete employee data
- Searching employee data

## Tech Stack

 - HTML
 - Tailwind CSS
- React.js


 



## Deployment

To deploy this project run

```bash
  npm install
  npm start
```
